# Double-Headed Growth — Geopolitical Intelligence Platform

**Research Paper:** "Double-Headed Growth: Evaluating Russia's Sanction Evasion Networks, Asymmetric Geopolitical Re-Routing, and Banking Loopholes"

**Author:** Makhzumul Islam Mahdi | ORCID: 0009-0007-1769-7771

## Platform Overview

A professional geopolitical intelligence and economic analytics platform visualizing all major findings from the research paper. Designed to meet the standards of major research organizations such as CSIS, Atlantic Council, and World Bank data platforms.

| Module | Page | Coverage |

## Run

```bash
streamlit run app.py --server.port 5000
```

## Stack

- **Streamlit** — UI framework
- **Plotly** — Interactive charts, Sankey diagrams, network graphs, geopolitical maps
- **Pandas** — Data management
- **NetworkX** — Network graph computation

## Data Sources

BIS, CBR (Bank of Russia), CREA, Eurasian Development Bank, Eurostat, NBKR, U.S. Treasury OFAC, CSIS, Reuters, Financial Times, AUTOSTAT
